# PyEPH - Python-based Package for Electron-Phonon Coupling Calculations

## Features
- **Phonon Dispersion, Electronic Band, and Electron-Phonon Coupling Calculations**: Compute phonon frequencies and eigenvectors along arbitrary q-paths, electronic band structures, and electron-phonon coupling matrix elements.
- **Charge Transport Properties**: Calculate mobility, optical conductivity, and spectral functions for both model and ab initio systems (interfacing with Quantum ESPRESSO and PERTURBO)
- **MPI Parallelization** using `mpi4py`
- **HDF5 Integration**
